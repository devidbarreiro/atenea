# HeyGen Video Generation

Generación de videos con avatares hablantes usando HeyGen.

## Descripción

HeyGen permite crear videos con avatares hablantes realistas que sincronizan perfectamente con el audio.

## Características

- ✅ Avatares hablantes realistas
- ✅ Sincronización labial perfecta
- ✅ Múltiples avatares y voces
- ✅ Fondos personalizables
- ✅ Videos transparentes (WebM)

## Precios

- **HeyGen Avatar V2**: 5 créditos por segundo
- **HeyGen Avatar IV**: 15 créditos por segundo
