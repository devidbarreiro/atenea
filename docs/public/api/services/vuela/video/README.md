# Vuela.ai Video Generation

Generación de videos con scripts personalizados usando Vuela.ai.

## Descripción

Vuela.ai permite crear videos desde scripts con personalización avanzada de estilo y animación.

## Características

- ✅ Generación desde scripts
- ✅ Personalización de estilo
- ✅ Múltiples modos de generación
- ✅ Validación de tokens
- ✅ Configuración avanzada

## Precio

- **Vuela.ai**: ~3 créditos por segundo (orientativo)
