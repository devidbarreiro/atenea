# Gemini Veo - Video Generation

Generación de videos cinematográficos de alta calidad usando Google Gemini Veo.

## Descripción

Gemini Veo permite crear videos desde texto o imágenes con calidad cinematográfica. Ideal para b-roll, paisajes y contenido visual general.

## Características

- ✅ Generación desde texto o imagen
- ✅ Múltiples resoluciones disponibles
- ✅ Soporte para extensión de video
- ✅ Máscaras y referencia de imágenes
- ✅ Último frame para continuidad

## Precio

- **Gemini Veo 2/3**: 50 créditos por segundo
- **Gemini Veo 3 + Audio**: 75 créditos por segundo
