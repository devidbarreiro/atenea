# Freepik Image Search

Búsqueda de imágenes, iconos y vectores en Freepik.

## Descripción

Busca y descarga imágenes, iconos y vectores de stock desde Freepik. Contenido gratuito y premium disponible.

## Características

- ✅ Búsqueda de imágenes e iconos
- ✅ Filtros por orientación y licencia
- ✅ Contenido gratuito y premium
- ✅ Descarga directa
- ✅ Integración con proyectos
