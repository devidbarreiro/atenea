# ElevenLabs Text-to-Speech

Conversión de texto a voz usando ElevenLabs.

## Descripción

ElevenLabs TTS permite convertir texto a voz natural con múltiples voces, idiomas y controles de estilo.

## Características

- ✅ Voces naturales y expresivas
- ✅ Múltiples idiomas soportados
- ✅ Control de velocidad, estabilidad y estilo
- ✅ Timestamps carácter por carácter
- ✅ Múltiples modelos disponibles

## Precio

- **ElevenLabs TTS**: 0.017 créditos por carácter
