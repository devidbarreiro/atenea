# OpenAI Sora - Video Generation

Generación de videos de alta calidad con efectos usando OpenAI Sora.

## Descripción

Sora permite crear videos desde texto o imagen con efectos especiales y escenas complejas. Ideal para contenido dinámico y creativo.

## Características

- ✅ Generación desde texto o imagen
- ✅ Múltiples modelos (Sora-2, Sora-2 Pro)
- ✅ Videos con efectos especiales
- ✅ Escenas complejas y dinámicas
- ✅ Múltiples resoluciones

## Precios

- **OpenAI Sora-2**: 10 créditos por segundo
- **OpenAI Sora-2 Pro**: 50 créditos por segundo
