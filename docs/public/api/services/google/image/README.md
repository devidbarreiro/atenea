# Gemini Image - Image Generation

Generación de imágenes usando Google Gemini Image.

## Descripción

Gemini Image permite crear imágenes desde texto, editar imágenes existentes o crear composiciones con múltiples imágenes.

## Características

- ✅ Generación desde texto
- ✅ Edición de imágenes (image-to-image)
- ✅ Composición con múltiples imágenes
- ✅ Múltiples relaciones de aspecto
- ✅ Control de estilo y calidad

## Precio

- **Gemini Image**: 2 créditos por imagen
