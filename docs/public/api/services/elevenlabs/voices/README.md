# ElevenLabs Voice Management

Gestión de voces disponibles en ElevenLabs.

## Descripción

Lista y gestiona las voces disponibles para usar en la generación de audio con ElevenLabs TTS.

## Endpoints Disponibles

- **[List Voices](list_voices.md)** - Listar todas las voces disponibles
- **[Get Voice](get_voice.md)** - Obtener detalles de una voz específica
