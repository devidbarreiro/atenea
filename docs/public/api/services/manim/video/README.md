# Manim Video Generation

Generación de videos animados con citas usando Manim.

## Descripción

Manim permite crear videos animados con citas textuales, ideal para contenido educativo y de redes sociales.

## Características

- ✅ Videos animados con citas
- ✅ Personalización de colores y fuentes
- ✅ Múltiples calidades (480p a 4K)
- ✅ Soporte para autor de la cita
- ✅ Duración automática o personalizada

## Precio

- **Manim**: ~1 crédito por video (independiente de duración)
